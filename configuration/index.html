Configuration

.simpspector.yml

You will need to put a configuration file `.simpspector.yml` in your projects root directory.

Gadgets

SimpSpector's static code analysis checks are build to be extendable und configurable. Checks are grouped in **SimpSpector Gadgets** or **gadgets** for short. You can find a list of all gadgets below, together with details on the specific configuration options all f these gadgets have (Config References).

Function Blacklist

Detect unsecure, dangerous or unwanted functions/statements. Error level can be configured individually.

Security Checker

Checks for known security vulnerabilities, using sensiolabs/security-checker.

Config Reference

# gadget: Function Blacklist
function_blacklist:
    files: src/
    blacklist:
        die: notice
        echo: notice
        var_dump: error


# gadget: Security Checker
security-checker:
  director: ./
  level: critical
